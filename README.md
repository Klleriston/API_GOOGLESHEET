# Manipulando API GoogleSheets

Projeto realizado em Node onde o foco é manipular uma planilha google via codigo, adicionando linhas e colunas.

## Instalações: 

Use o pacote padrão NPM mais os do google (google-spreadsheet e google-auth-library)
 
```bash
npm init -y 
npm install google-spreadsheet google-auth-library node-fetch
```

## Para rodar o codigo:
```node
node app.js

```
